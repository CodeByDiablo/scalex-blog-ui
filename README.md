## Design Approach

This project follows a modern SaaS/frontend design philosophy using:

- Flexbox layouts
- Component-based UI structure
- Large viewport typography (`vw`)
- Dark mode aesthetics
- Card-based content system
- Minimal spacing & hierarchy
- Modern hover interactions
